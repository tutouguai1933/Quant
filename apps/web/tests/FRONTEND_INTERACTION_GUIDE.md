# 前端界面交互动线指南

## 设计原则
- **决策页**：有明确的主动作按钮，点击后触发动作或跳转
- **配置页**：首屏是摘要，点击"查看详情"打开抽屉，修改配置后保存
- **工具页**：只查明细，返回入口带回到决策页

---

## 1. 首页 `/` - 决策驾驶舱

### 主要点击动线

| 点击位置 | 预期结果 |
|----------|----------|
| **"运行 Qlib 信号流水线" 按钮** | 触发研究训练 → 进度条显示 → 完成后候选更新 |
| **"查看研究报告" 按钮** | 跳转到 `/signals` 信号页 |
| **"启动策略" 按钮** | 触发策略启动动作（需登录） |
| **"派发最新信号" 按钮** | 触发信号派发动作（需登录） |
| **"去策略页确认执行" 按钮** | 跳转到 `/strategies` 策略页 |
| **"去任务页处理自动化" 按钮** | 跳转到 `/tasks` 任务页 |

### 交互动线
```
首页 → 运行研究 → 看进度 → 完成后去信号页看候选
首页 → 去策略页 → 看执行器 → 派发信号
首页 → 去任务页 → 看自动化状态 → 控制动作
```

## 6. 因子页 `/features` - 配置页 ⚠️ 动线不清晰

### 当前设计问题
- ❌ **没有主动作按钮**：用户不知道点击什么能触发下一步
- ❌ **只有"查看详情"抽屉**：看完详情后不知道去哪
- ❌ **配置保存后没有反馈**：保存成功后没有引导用户去研究页运行

### 主要点击动线

| 点击位置 | 预期结果 |
|----------|----------|
| **"查看分类详情" 按钮** | 打开抽屉 → 显示因子分类清单 |
| **"查看启用详情" 按钮** | 打开抽屉 → 显示主判断/辅助因子 |
| **"查看因子配置" 按钮** | 打开抽屉 → 显示因子预设选择器 + 因子勾选 |
| **"查看有效性详情" 按钮** | 打开抽屉 → 显示各类因子有效性 |
| **"查看冗余详情" 按钮** | 打开抽屉 → 显示因子冗余分析 |
| **"保存配置" 按钮（抽屉内）** | 保存配置 → 页面刷新（无明确反馈） |

### 缺失的动线
```
❌ 因子页 → 调整因子权重 → ？？？没有引导去研究页运行
❌ 因子页 → 保存配置 → ？？？没有反馈告知下一步
```

### 建议补充的动线
```
✅ 因子页 → 调整因子权重 → 点击"保存并运行研究" → 去信号页看结果
✅ 因子页 → 保存配置 → 显示"配置已保存，去研究页运行看效果" → 提供跳转按钮
```

## 完整流程动线

### 研究到执行流程
```
首页 → 运行研究 → 信号页（看候选） → 评估页（看推荐） → 策略页（确认执行） → 任务页（看自动化）
```

### 配置调整流程（当前不清晰）
```
因子页 → 调整因子 → ？？？ → 研究页 → 调整参数 → ？？？ → 去信号页运行
```

### 工具页查看流程
```
策略页 → 点击余额 → 余额页（看明细） → 点击回到策略页
策略页 → 点击订单 → 订单页（看明细） → 点击回到策略页
```

## 动线问题总结

| 问题 | 影响页面 | 建议修复 |
|------|----------|----------|
| **没有主动作按钮** | 因子页、研究页、数据页、回测页 | 添加"保存并运行研究"按钮 |
| **保存后没有反馈** | 因子页、研究页、数据页、回测页 | 保存成功后显示"去信号页运行"按钮 |
| **不知道去哪运行** | 因子页、研究页 | 添加跳转到信号页的入口按钮 |
| **动线断裂** | 配置页 → 运行页 | 添加明确的"下一步去哪"指引 |
