## 后续方向

### 近期

| 方向 | 说明 |
|------|------|
| 模型效果持续追踪 | 积累更多实盘样本后评估 Brier Score |
| A/B 统计显著性 | 样本充足后评估 ML vs 启发式是否有统计显著差异 |
| Gate 阈值调优 | 根据实盘表现调整 ML Live Gate 阈值 |

### 中期

| 方向 | 说明 |
|------|------|
| 多标的组合优化 | Top-K 选币替代当前 TOP1 |
| 特征漂移检测 | 监控特征分布变化，自动触发重训练 |
| 回测系统增强 | 更完善的多周期回测对比 |

### 长期

| 方向 | 说明 |
|------|------|
| 多交易所支持 | 扩展到 OKX、Bybit 等 |
| RL 策略优化 | 增强学习优化策略参数 |
| 高频因子 | 分钟级因子和预测 |
