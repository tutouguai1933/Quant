## 当前最重要的判断标准

### 什么算 phase2 完成

以下 5 条都满足，才算 phase2 真正结束：

1. 你能在页面里完整配置一轮研究
2. 你能清楚比较至少两轮实验
3. 你能一眼看懂为什么推荐、为什么淘汰
4. 你能看懂研究结果和执行结果差在哪里
5. 自动化出问题时，页面会明确告诉你怎么处理
